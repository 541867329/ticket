12306抢票客户端-PyQt5实现
=============================

安装依赖：
--------

```
pip install requests peewee 
```
或
```
easy_install requests peewee
```

关于PyQt5
---------
经测试可以在Mac, Linux上运行, 由于Windows版PyQt5没有编译OpenGL, 如果想
在Windows测试, 需要下载源码进行编译安装

关于Python
----------
Python 2.7最佳, 兼容Python 3.x
